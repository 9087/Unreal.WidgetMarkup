WidgetMarkup 用户手册

使用简洁的 XML 在 Unreal Engine 中构建 UMG 控件树，并通过响应式 Python 组件驱动运行时逻辑。

本手册覆盖 WidgetMarkup 编辑器插件及其配套插件 WidgetMarkupPythonIntegration，默认你已具备 Unreal Editor 与 UMG 的基础使用经验。

目录

  1. 快速开始启用插件、创建第一个 markup 文件并打开实时预览。
  2. XML 语法与属性系统元素模型、属性路径、类型转换器与 Name 属性。
  3. WidgetBlueprint、变量与样式表声明蓝图变量、实现接口、挂载样式表。
  4. Python 集成概览安装集成插件并将 Python 组件绑定到 UserWidget。
  5. 响应式 Python 组件使用 @reactive、@computed 与 ObservableCollection。
  6. 排错与参考编辑器命令、常见报错与源码定位。
