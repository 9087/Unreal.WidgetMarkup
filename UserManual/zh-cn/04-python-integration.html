4. Python 集成概览

通过 Script 属性将 Python 组件绑定到 WidgetMarkup 生成的 UserWidget。

4.1 工作机制

  1. 读取 Script 模块名。
  2. 导入模块并查找同名类。
  3. 调用 WidgetMarkupComponent.create 实例化。
  4. 把实例绑定到 UserWidget 扩展。

4.2 示例

<WidgetBlueprint Script="my_game.ui.HealthBar">...</WidgetBlueprint>
