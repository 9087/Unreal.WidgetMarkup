2. XML 语法与属性系统

元素映射到 UMG 类型，属性名映射到 PropertyPath，字符串通过转换器写入原生值。

2.1 元素模型

  • WidgetBlueprint：根节点并在结束时编译。
  • WidgetTree：设置根控件。
  • PanelWidget：可包含多个子控件。
  • ContentWidget：只允许一个子控件。

2.2 属性路径

  • Property
  • Outer.Inner
  • Slots[0].Content
  • Slot.LayoutData.Alignment
<TextBlock Text="Hello" Slot.LayoutData.Alignment="0.5,0.5" />

2.3 Name 属性

设置 Name 后，控件会以同名变量暴露到生成蓝图类。Name 必须符合蓝图变量命名规则且在同一蓝图内唯一。
